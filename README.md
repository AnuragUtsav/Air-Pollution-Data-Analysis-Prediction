# Air Pollution Data Analysis & Prediction

## Overview
This project focuses on analyzing air pollution data and building a prediction model using Python. It includes data preprocessing, exploratory data analysis (EDA), visualization, and regression techniques to predict pollution levels.

---

## Objectives
- Understand air pollution trends
- Perform data cleaning and preprocessing
- Visualize key patterns in pollution data
- Build a prediction model using Linear Regression
- Evaluate model performance using metrics like R² score

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
